## :checkered_flag: Getting starter


```bash

# Clone the project

$ git clone https://github.com/JoaoGomes5/vue-query

# Go to project directory

$ cd vue-query

# Install the dependencies

$ npm install 

# Run the project

$ npm run dev 

# The server will initialize on <http://localhost:5173/>

```
